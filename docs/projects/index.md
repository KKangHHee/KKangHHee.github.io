# 프로젝트 포트폴리오

실무 수준의 팀 프로젝트 경험을 통해 **백엔드 개발**, **성능 최적화**, **보안 구현** 역량을 쌓아왔습니다.

## 주요 프로젝트

### 1️⃣ Bargain Hunter (2025.07 ~ 2025.10)

**국내 관광 지원 서비스 - MSA 기반 풀스택 개발**

- **팀 구성**: BE 4명, FE 1명
- **주요 역할**:
  - OAuth2 + JWT 기반 인증 서버 개발
  - Redis 기반 이메일 인증 시스템 구축
  - Gateway 라우팅 및 인증 토큰 전달 구조 설계
- **주요 성과**:
  - 이메일 발송 이벤트 기반 아키텍처 적용 → **API 응답 속도 92% 개선**
  - PKCE 적용으로 OAuth2 보안 강화
  - Redis Hash + HINCRBY로 동시성 문제 해결

[자세히 보기 →](/docs/projects/bargain-hunter/overview)

### 2️⃣ 보안 티켓 관리 플랫폼 (2025.04 ~ 2025.05)

**정보보호 컴플라이언스(ISMS-P, ISO27001) 점검 플랫폼**

- **팀 구성**: BE 8명, FE 3명
- **주요 역할**:
  - 세션 기반 인증/인가 구조 설계 및 구현
  - 사용자 관리 CRUD 및 페이징 API 구현
  - MyBatis 활용 복합 검색/정렬 최적화
- **주요 성과**:
  - 관리자 페이지 복합 **검색 성능 36% 개선** (24.45ms → 15.51ms)
  - 세션 기반 보안 강화 및 역할 기반 접근 제어 구현
  - 팀 협업 효율 상승을 위한 API 표준화

[자세히 보기 →](/docs/projects/security-ticket/overview)

## 기술적 강점

### 🔐 보안 및 인증

- JWT + Refresh Token 관리
- OAuth2 + PKCE 플로우
- 세션 기반 인증/인가
- CSRF/XSS 대응

### ⚡ 성능 최적화

- 이벤트 기반 아키텍처 (API 응답 92% 개선)
- MyBatis 동적 쿼리 최적화 (검색 성능 36% 개선)
- Redis 기반 캐싱 전략
- 동시성 제어 (HINCRBY 활용)

### 🏗️ 아키텍처 설계

- MSA 기반 서비스 구조
- Gateway를 통한 라우팅 및 인증
- 이벤트 기반 비동기 처리
- 하이브리드 ORM 전략 (JPA + MyBatis)

### 🤝 협업 및 커뮤니케이션

- API 명세서 작성 및 표준화
- 코드 리뷰 문화 구축
- 애자일 스프린트 운영
- 팀 간 기술 공유
