# 🔐 Security Ticket

**정보보호 컴플라이언스(ISMS-P, ISO27001) 점검을  
Excel·이메일 중심의 수동 프로세스에서 벗어나  
웹 기반으로 디지털 전환한 보안 검증 관리 플랫폼입니다.**

---

## 📌 프로젝트 한 줄 요약

> **보안 컴플라이언스 점검 프로세스를  
> 역할 기반 권한 관리와 자동화된 검증 흐름으로 표준화한 웹 플랫폼**

---

## 🎯 핵심 문제 인식

- Excel / 이메일 기반 점검으로 **이력 관리 어려움**
- 검증 진행 상태 파악 불가
- 역할(관리자/고객사)에 따른 접근 제어 미흡
- 반복적인 수동 커뮤니케이션으로 업무 비효율 발생

---

## 🚀 해결 방향

- **웹 기반 보안 점검 프로세스 표준화**
- 역할 기반 접근 제어 및 세션 관리
- 검증 상태 시각화 및 자동 알림
- 복잡한 검색/조회 기능의 성능 최적화

---

## 🧩 주요 구현 포인트

- 커스텀 **세션 기반 인증/인가 구조**
- JPA + MyBatis **하이브리드 ORM 전략**
- 복합 검색 쿼리 성능 개선 (최대 TPS 4배 향상)
- 팀 단위 협업을 고려한 API 표준화
